### :hamburger: Finalidade do projeto
- Projeto criado com o objetivo de conhecer e praticar o framework Vue.js <img src='https://github.com/Riquecelo/Projeto-Vue/blob/main/public/faviconVue.ico'/>
- Simular o registro e a visualização de pedidos de um site
- O cliente realiza seu pedido na página principal
- Os pedidos são visualizados na página administrativa 'Pedidos'

<img src='https://github.com/Riquecelo/Projeto-Vue/blob/main/public/img/show.gif'/>

##  Para executar o projeto

### :hamburger: Execute o seguinte comando no terminal
```
npm run serve
```

<img src='https://github.com/Riquecelo/Projeto-Vue/blob/main/public/img/start-npm-run-serve.gif' height='350' width='950'/>

<hr/>

### :hamburger: Execute o seguinte comando em outro terminal
```
npm run backend
```

<img src='https://github.com/Riquecelo/Projeto-Vue/blob/main/public/img/start-npm-run-backend.gif' height='350' width='950' />
